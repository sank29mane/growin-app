# Plan 24-01: Reasoning Trace UI Design & State Binding

Implement a visual "Thinking" trace for agentic reasoning in the SwiftUI dashboard with auto-collapse, expandable grids, and 120Hz/Accessibility polish.

## 1. Research Sync (Updated via NotebookLM)
*   **SF Symbols**: Use `.symbolEffect(.variableColor.iterative, isActive: isThinking)` for indefinite states and `.symbolEffect(.bounce, value: thoughtCount)` for discrete events.
*   **Accessibility**: Use `.accessibilityLiveRegion(.polite)` and the `.updatesFrequently` trait to prevent VoiceOver flooding during streaming.
*   **Transitions**: Implement `SlotModifier` for a kinetic vertical "Slot Machine" entry (offset: 20 -> 0, opacity: 0 -> 1). Use `.scrollTransition` for smooth roll-in/roll-out.
*   **Performance**: Utilize the `@Equatable` macro and the `.equatable()` view modifier to bypass reflection-based diffing, maintaining the 120Hz frame budget.

## 2. Models & State Management
*   [ ] **Models.swift**: Add `reasoningSteps: [ReasoningStep]?` to `ChatMessageModel`.
*   [ ] **ChatViewModel.swift**: 
    *   Update `sendMessageAsync` to persist `activeReasoningSteps` into the `assistantModel` when `isProcessing` becomes `false`.
    *   Ensure `activeReasoningSteps` is cleared only after being saved to the message.

## 3. UI Component Evolution
*   [ ] **ChatReasoningTraceView.swift**:
    *   Add `@State private var isExpanded = false`.
    *   Implement **Collapsed View**: A single "Intelligence Verified" chip (or a summary row with "+N more" if steps > 3).
    *   Implement **Expanded View**: A `LazyVGrid` (2 columns) showing all `ReasoningStepChip` items.
    *   Implement **SlotModifier**: Create a view modifier for the vertical roll-in transition.
    *   Add **SF Symbol Animations**: Use `.symbolEffect` on icons based on agent status.
    *   Add **Numeric Transitions**: Use `.contentTransition(.numericText())` for confidence scores.

## 4. Integration
*   [ ] **ChatBubble**: Update to render `IntelligenceTraceView` (which will use the new `ChatReasoningTraceView` logic) if `message.reasoningSteps` is present.
*   [ ] **ChatView**: Ensure smooth scrolling using `ScrollViewReader` when the trace expands/collapses.
