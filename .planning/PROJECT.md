# PROJECT: Growin

## Vision
Transform Growin into a self-correcting trading weapon for LSE Leveraged ETFs, optimized for M4 Pro (48GB RAM) and high-fidelity local profit extraction. The platform combines hardware-aware MAS (Multi-Agent System) with a radical Sovereign Ledger UI.

## Tech Stack
- **Frontend**: macOS Native SwiftUI (120Hz), Stitch MCP (UI Generation)
- **Backend**: Python 3.11+, FastAPI, uv (Package Management)
- **AI/ML**: Native MLX (Apple Silicon GPU), Neural JMCE (NPU/ANE), vllm-mlx
- **Data**: Trading 212 MCP, Alpaca Primary, DuckDB (Analytics)
- **Hardware Target**: Apple M4 Pro / Max / Ultra

## Core Milestones
1. **v1.0: Foundation & Precision Alpha** (Complete) - Core orchestration and financial precision.
2. **v2.0: SOTA Intelligence & Financial Autonomy** (Complete) - professionalization and MAS evolution.
